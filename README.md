## AS自动排班助理

本项目实现了基于Python3和HTTP交互的AS自动排程。

在使用本项目前，您可能需要先安装Python3环境。  
以及使用pip安装以下的包：

```shell
pip install requests beautifulsoup4 html5lib
```

本项目初步实现了以下功能：

- 基于自定义往返机场、服务、价格的单航机单航线自动排程
- 基于配置解析的单航机多航线自动排程
- 自动纠正时刻表异常并发出提醒
